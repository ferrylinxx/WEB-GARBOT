## ¿Qué es un Prompt?

Un **prompt** es la instrucción o pregunta que le das a la IA. Es como una receta: cuanto más específica y clara, mejor será el resultado.

### Anatomía de un Prompt Perfecto

```
[CONTEXTO] + [TAREA] + [FORMATO] + [RESTRICCIONES] + [EJEMPLOS]
```

## Principios Fundamentales

### 1. Sé Específico

❌ **Malo**: "Escribe sobre marketing"

✅ **Bueno**: "Escribe un artículo de 500 palabras sobre estrategias de marketing digital para pequeñas empresas en 2025, enfocándote en redes sociales y SEO"

### 2. Proporciona Contexto

❌ **Malo**: "Crea un email"

✅ **Bueno**: "Soy el CEO de una startup tecnológica. Crea un email para invitar a inversores a nuestra ronda Serie A, destacando nuestro crecimiento del 300% en 6 meses"

### 3. Define el Formato

❌ **Malo**: "Dame información sobre Python"

✅ **Bueno**: "Crea una tabla comparativa de las 5 principales librerías de Python para análisis de datos, incluyendo: nombre, uso principal, curva de aprendizaje y popularidad"

### 4. Establece Restricciones

❌ **Malo**: "Explica la fotosíntesis"

✅ **Bueno**: "Explica la fotosíntesis en máximo 100 palabras, usando analogías simples, para un estudiante de 12 años"

### 5. Usa Ejemplos

❌ **Malo**: "Escribe un eslogan"

✅ **Bueno**: "Escribe 5 eslogans para una app de meditación, similar a 'Just Do It' de Nike - cortos, memorables e inspiradores"

## Técnicas Avanzadas

### Técnica 1: Role Playing

Asigna un rol específico a la IA:

```
Actúa como un experto en ciberseguridad con 15 años de experiencia.
Explica qué es un ataque de phishing y cómo prevenirlo,
dirigido a empleados de una empresa sin conocimientos técnicos.
```

### Técnica 2: Chain of Thought (Cadena de Pensamiento)

Pide que muestre el razonamiento:

```
Resuelve este problema paso a paso, mostrando tu razonamiento:
Si una empresa tiene 100 clientes y crece 20% mensual,
¿cuántos clientes tendrá en 6 meses?
```

### Técnica 3: Few-Shot Learning

Proporciona ejemplos del resultado deseado:

```
Convierte estas descripciones en títulos atractivos:

Descripción: "Aplicación para hacer listas de tareas"
Título: "TaskMaster: Organiza tu Vida en Segundos"

Descripción: "Servicio de entrega de comida rápida"
Título: "FlashFood: Del Restaurante a tu Puerta en 20 Minutos"

Ahora hazlo con:
Descripción: "Plataforma de aprendizaje de idiomas"
Título: [tu respuesta]
```

### Técnica 4: Iteración Progresiva

Refina gradualmente:

```
1. "Dame 10 ideas de nombres para una app de fitness"
2. "De esas 10, dame las 3 mejores con explicación"
3. "Toma la mejor y crea 5 variaciones"
4. "Para la variación más fuerte, crea un logo conceptual"
```

### Técnica 5: Constraints Creativos

Usa limitaciones para fomentar creatividad:

```
Escribe una historia de ciencia ficción de exactamente 50 palabras,
que incluya: un robot, una tormenta y un secreto,
sin usar las palabras "dijo", "fue" o "había"
```

## Plantillas por Caso de Uso

### Para Escritura de Contenido

```
Tema: [tu tema]
Audiencia: [quién lo leerá]
Tono: [profesional/casual/técnico/etc]
Longitud: [número de palabras]
Objetivo: [informar/persuadir/entretener]
Incluir: [puntos clave específicos]
Evitar: [temas o palabras a evitar]
```

### Para Análisis de Datos

```
Analiza estos datos: [tus datos]
Busca: [patrones/tendencias/anomalías]
Presenta resultados en: [tabla/gráfico/lista]
Incluye: [insights accionables]
Enfócate en: [aspecto específico]
```

### Para Código

```
Lenguaje: [Python/JavaScript/etc]
Tarea: [qué debe hacer el código]
Input: [qué recibe]
Output: [qué debe devolver]
Restricciones: [librerías permitidas, complejidad]
Incluye: [comentarios/tests/documentación]
```

### Para Creatividad

```
Tipo: [logo/slogan/nombre/etc]
Para: [producto/servicio/empresa]
Estilo: [moderno/clásico/minimalista]
Inspiración: [referencias que te gustan]
Evitar: [estilos que no quieres]
Cantidad: [número de opciones]
```

## Errores Comunes y Soluciones

### Error 1: Prompts Demasiado Vagos

❌ "Ayúdame con mi negocio"

✅ "Analiza mi plan de negocio de una cafetería en Madrid y sugiere 3 estrategias para diferenciarnos de Starbucks"

### Error 2: Asumir Conocimiento Previo

❌ "Continúa con lo que hablamos"

✅ "Basándote en nuestra conversación anterior sobre marketing digital (donde discutimos SEO y SEM), ahora profundiza en estrategias de contenido"

### Error 3: No Iterar

❌ Aceptar la primera respuesta siempre

✅ "Esto está bien, pero hazlo más conciso y añade ejemplos prácticos"

### Error 4: Prompts Demasiado Complejos

❌ Un párrafo de 300 palabras con 15 requisitos diferentes

✅ Divide en múltiples prompts secuenciales

### Error 5: No Especificar el Formato

❌ "Dame información sobre competidores"

✅ "Crea una tabla comparativa de 5 competidores con columnas: nombre, precio, características principales, ventajas, desventajas"

## Optimización de Prompts

### Método CLEAR

- **C**ontext (Contexto): Proporciona background
- **L**ength (Longitud): Especifica extensión
- **E**xamples (Ejemplos): Muestra qué quieres
- **A**udience (Audiencia): Define para quién es
- **R**estrictions (Restricciones): Establece límites

### Checklist de Calidad

Antes de enviar tu prompt, verifica:

- [ ] ¿Es específico y claro?
- [ ] ¿Incluye contexto suficiente?
- [ ] ¿Define el formato deseado?
- [ ] ¿Establece restricciones necesarias?
- [ ] ¿Proporciona ejemplos si es relevante?
- [ ] ¿Especifica la audiencia?
- [ ] ¿Indica el tono deseado?
